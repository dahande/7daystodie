# 7 Days to Die - AEC Mods Collection

このリポジトリは、**7 Days to Die** のゲーム用の高度なモッドコレクションです。Aeclipse によって開発されています。

## 📋 含まれるモッド

### 🎮 AEC AIO BOSS EXTREME EDITION
- **バージョン**: 3.0.2
- **説明**: 極端に難しいボスエディション
- **特徴**:
  - ゲーム内のボスエンカウンターを大幅に強化
  - サーバーサイドのみ対応
  - 極限の難易度でプレイを楽しむプレイヤー向け

### 🎯 AEC AIO BOSS X EXTENDED QUESTS
- **説明**: 拡張クエストシステム
- **特徴**:
  - ボスコンテンツとの相互作用を拡張
  - より多くのクエスト関連機能を提供

### ⚔️ AEC ENDGAME OVERHAUL
- **バージョン**: 3.0.4
- **説明**: エンドゲームコンテンツの大幅なリワーク
- **特徴**:
  - ゲーム後半のコンテンツを全面的に改造
  - サーバーサイドのみ対応
  - より充実したエンドゲーム体験を提供

### 🌍 AEC Project Z JP Localization
- **説明**: 日本語ローカライゼーション
- **特徴**:
  - 上記のモッドを日本語対応にします
  - 日本のプレイヤーに最適なゲーム体験を提供

## 🚀 インストール

1. このリポジトリをクローンします
2. 7 Days to Die のモッドフォルダに各モッドフォルダをコピーします
3. ゲームを起動してモッドが読み込まれることを確認します
